Esse projeto requer o node.js

Para executar esse projeto devem ser seguidos os seguintes passos no cmd:

1. Ir para o caminho raiz do projeto
2. npm install -g crud-file-server
3. npm install crud-file-server
4. Ir para o caminho "cd node_modules/crud-file-server/example"
5. Inserir o comando crud-file-server -p 8000
